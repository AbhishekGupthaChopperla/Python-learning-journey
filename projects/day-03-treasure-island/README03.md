# 🏝️ Treasure Island Game

A simple text-based adventure game built with Python as part of the **100 Days of Code: Python Bootcamp (Day 3 Project)**.

The goal of the game is to find the hidden treasure by making the correct choices. One wrong move and it's **Game Over!**

---

## 🎮 How to Play

The player is presented with a series of choices:

1. Choose **left** or **right**
2. Choose **wait** or **swim**
3. Choose a door: **red**, **blue**, or **yellow**

Only one path leads to the treasure!

### Winning Path

```text
left → wait → yellow
```
