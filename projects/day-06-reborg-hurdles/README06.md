# 🤖 Reeborg's World Hurdles

A collection of hurdle-solving programs built with Python as part of the **100 Days of Code: Python Bootcamp (Day 6 Project)**.

The goal of these challenges is to help Reeborg navigate different obstacle courses by using functions, loops, and conditional logic.

---

## 🎮 How to Play

The player writes Python code to control Reeborg and guide the robot to the goal.

The challenges include:

1. Hurdle 1 – Jump over fixed obstacles
2. Hurdle 2 – Use a loop to solve repeated hurdles
3. Hurdle 3 – Handle hurdles with different positions
4. Hurdle 4 – Navigate obstacles using conditional logic
5. Maze - Go to end avoiding all the ends and obstacles

### Example

```text
Move Forward
Jump Over Hurdle
Continue Until Goal Reached
```
