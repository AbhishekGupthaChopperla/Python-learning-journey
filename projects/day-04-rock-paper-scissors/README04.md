# 🎮 Rock Paper Scissors Game

A simple Rock Paper Scissors game built with Python as part of the **100 Days of Code: Python Bootcamp (Day 4 Project)**.

The goal of the game is to defeat the computer by choosing the correct move.

---

## 🎮 How to Play

The player chooses:

* **0** for Rock 🪨
* **1** for Paper 📄
* **2** for Scissors ✂️

The computer randomly selects its move, and the winner is determined based on the game rules.

### Winning Rules

```text
Rock beats Scissors
Scissors beats Paper
Paper beats Rock
```

## 🚀 Features

* User choice input

* ASCII art for Rock, Paper, and Scissors
* Win, Lose, and Draw outcomes
