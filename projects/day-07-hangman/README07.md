# 🎮 Hangman Game

A classic Hangman game built with Python as part of Day - 07 of the **100 Days of Code: Python Bootcamp**.

The game randomly selects a word, and the player must guess it one letter at a time before running out of lives.

---

## 🚀 Features

* Random word selection
* Letter-by-letter guessing
* Life tracking system
* ASCII Hangman graphics
* Win and lose conditions
* Duplicate guess detection

---

## 📂 Project Structure

```text
├── game.py
├── hangman_art.py
└── hangman_words.py
```

### game.py

Contains the main game logic, user input handling, and game flow.

### hangman_art.py

Stores the Hangman ASCII art stages and game logo.

### hangman_words.py

Contains the list of words used for random selection.

---

## 🎯 How to Play

1. Run the game.
2. Guess one letter at a time.
3. Correct guesses reveal letters in the hidden word.
4. Incorrect guesses reduce your lives.
5. Win by guessing the entire word before losing all lives.
